# How to calibrate the ESP Tamiya TBLE-04S

Switch the ESP on.

With the throttle at neutral (cmd_vel = 0.0), press the set button and release it when the LED is red. 

This will set the neutral instantly (Flashing red LED). 

Move the throttle to full throttle (cmd_vel = 1.0) then press the button again. The highpoint was set (Double flash Red). 

Then move the throttle to full brake (cmd_vel = -1.0) then press the set button. This sets the brake point. (LED will switch off).
